Common Abbreviations

O termo HTML refere-se a uma linguagem de marcação usada para criar páginas web. Ela é uma das principais linguagens da web.

Outras abreviações comuns incluem:

  • CSS - Usado para estilizar páginas web.
  • WWW - A rede global de computadores interconectados.
  • JSON - Um formato leve de troca de dados.
  • Fla - Abreviação de Flamengo
